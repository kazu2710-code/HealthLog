# HealthLog

時系列で健康状態を記録するための個人用PWAアプリです。

医師への受診時に「いつ・どんな症状が出たか」を正確に伝えるためのログツールとして作りました。AIによる診断ではなく、**記録した内容をAIに渡してサマリーを作る**ための橋渡しが目的です。

---

## これは個人用です

このリポジトリは個人の用途に特化して作られており、プロファイル（ボタン構成・スケジュール・タグ等）は作者本人のものが含まれています。

一般配布を想定したものではありませんが、HTMLファイル1枚で動く構造なので、興味があれば自由に使ってください。

---

## 使い方（最低限）

1. `healthlog.html` をブラウザで開く（またはホーム画面にインストール）
2. `[…]` メニュー →「プロファイルを読み込む」でJSONファイルを読み込む
3. ボタンをタップして記録、`[…]` →「ログを保存」でテキスト書き出し

プロファイルJSONを自分用に編集することで、ボタン構成・スケジュール・タグ色などをカスタマイズできます。

---

## 技術メモ

- HTML / CSS / JavaScript 単一ファイル
- データ保存：localStorage のみ（サーバー不要）
- 外部ライブラリ：なし
- PWA対応（manifest.json + Service Worker）
